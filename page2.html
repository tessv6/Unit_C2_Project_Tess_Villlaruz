Essential HTML elements

There are many HTML tags. However, there are some that are essential to every document. These include the document type declaration and the HTML tag itself. We will explain those here.


  • The Doc Type declaration goes at the beginning of every HTML document, and it lets the browser know to expect an HTML document. This is a Doc Type declaration:
    <!DOCTYPE html>.
  • The HTML tag indicates the beginning and end of the document's code. The HTML tag is this: <html>.
  • The "body" tag contains all of the actual contents of the document, rather than information that only coders, web browsers, and search engines can see. <body>.
  • The <head"> tag contains metadata (see Lesson 5: Metadata). This information is NOT shown in the document itself. This is only for computers and coders to view.
  • The <title> tag is the name of the document. It is not visible on the document itself. It apears in the browser title bar.
